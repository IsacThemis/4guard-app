# 🦅 MANUAL PARA COLABORADORES: BIENVENIDO A PROMETEO TITAN

Este manual es una guía sencilla para que puedas trabajar con **Prometeo TITAN** sin complicaciones. Piensa en TITAN como un "Asistente Inteligente Avanzado" que te ayudará a que todo el código sea perfecto y esté ordenado.

---

## 📍 1. ¿Qué es TITAN? (En palabras simples)
TITAN no es solo una carpeta con archivos, es como una **Mente Digital** dividida en tres partes:

1.  **El Cerebro (`core`)**: Donde TITAN piensa y procesa la información.
2.  **La Mochila de Habilidades (`skills`)**: Herramientas especiales que TITAN usa para diferentes tareas (como saber de leyes, de diseño o de organización).
3.  **El Puente (`.opencode`)**: Un túnel que permite que TITAN comparta sus ideas con otros programas como OpenCode.

---

## 🛠️ 2. Preparando tu Computadora
Antes de empezar, necesitamos que tu PC hable el mismo idioma que TITAN. Solo necesitas instalar estos 4 programas (dale "Siguiente" a todo durante su instalación):

1.  **Node.js**: El motor que hace que TITAN despierte.
2.  **Docker**: La "caja de memoria" donde TITAN guarda sus recuerdos.
3.  **Python**: El traductor que ayuda al "Puente" a funcionar.
4.  **Antigravity**: Tu libreta digital para ver y editar el proyecto.

---

## 🚀 3. Cómo poner a TITAN a trabajar (Paso a Paso)

### Paso A: Descargar el Proyecto
Baja la carpeta del proyecto a tu computadora y ábrela con Antigravity.

### Paso B: Encender la Memoria
Abre una terminal (Presiona `Ctrl + Ñ` en Antigravity) y escribe esto para prender los "recuerdos" de TITAN:
```bash
docker-compose up -d
```

### Paso C: Enseñar a TITAN a pensar
Escribe estos dos comandos para que TITAN prepare su cerebro:
```bash
cd core
npm install
npm run build
```

### Paso D: Conectar el Puente
Para que todo funcione unido, busca un archivo llamado **`Sync-Skills.bat`** y dale doble clic. Esto hará que todas las piezas encajen.

---

## 🧠 4. ¿Qué puede hacer TITAN por ti?
TITAN tiene "Súper Poderes" (Skills) que puedes usarle pidiéndolo directamente:
- **Reglas de Oro (Workflow)**: Evita que cometas errores al escribir código.
- **Súper Planificador**: Te ayuda a organizar tareas difíciles paso a paso.
- **Buscador Inteligente**: Puede leer documentos y responderte dudas complejas.

---

## 🛡️ 5. Reglas de Seguridad (¡Muy importante!)
Para que el proyecto nunca se rompa, solo recuerda dos cosas:

1.  **Antes de terminar**: Escribe `/audit` en el chat. TITAN revisará que todo esté limpio y ordenado.
2.  **Punto de Guardado**: Escribe `/git-checkpoint` antes de hacer cambios grandes. Es como guardar partida en un videojuego; si algo sale mal, podemos volver atrás.

---

## 🤝 6. ¿Necesitas ayuda?
Si ves algún error, lo más fácil es ejecutar el "Doctor":
- Dale doble clic al archivo `Sync-Skills.bat` de nuevo.
- O escribe en el chat: "¿TITAN, estás operativo?" y él mismo se revisará.
