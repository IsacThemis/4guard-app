# TITAN Skills Matrix (Catálogo Maestro)

Este es el índice central de habilidades disponibles para el agente. Lee las descripciones para decidir qué skill cargar en tu contexto.

## 🏛️ BLUEPRINTS (Arquitecturas Base)
- **`BLUEPRINTS/clean-architecture/SKILL.md`**: Enterprise Blueprint para proyectos backend. Usa Fastify, Prisma, TypeScript. Úsalo para crear la estructura base de un proyecto.

## 📐 WORKFLOW (Gobernanza y Reglas)
- **`WORKFLOW/governance-protocol.md`**: Interceptor de intención. **SIEMPRE ACTIVO**. Contiene las reglas de Deuda Técnica Cero y el protocolo de lectura del `NEXUS_BRIDGE.json` para evitar colisiones.

## 🌟 COMMUNITY SKILLS (Nuevas Adquisiciones)
- **`using-superpowers/SKILL.md`**: Habilidades avanzadas de superusuario y automatización.
- **`planning-with-files/SKILL.md`**: Gestión de tareas y planeación basada en archivos (task_plan.md).
- **`notebooklm-skill/SKILL.md`**: Integración profunda con Google NotebookLM para respuestas basadas en fuentes.
- **`skillx/SKILL.md`**: Motor de búsqueda e instalación quirúrgica de nuevas habilidades (Cherry-Pick).

## 🛡️ GOVERNANCE (Auditoría y Reglas)
- **`PolicyAuditDoc/SKILL.md`**: Audita archivos Rego/Cedar y genera documentación legible para auditoría de riesgos.
- **`RuleSynthesizer/SKILL.md`**: Sintetiza reglas de comportamiento y gobernanza a partir de requisitos funcionales.
